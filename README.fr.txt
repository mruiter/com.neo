Notes de support :

La façon la plus fiable de mettre à jour les appareils alimentés par batterie :  
1. Placez le capteur près de Homey (< 1 mètre)  
2. Changez les paramètres selon vos préférences  
3. Réveillez le capteur (triple clic sur le bouton)  
4. Pendant le clignotement de la LED (indiquant la connexion à Homey), appuyez sur "enregistrer les paramètres"

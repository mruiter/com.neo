Unterstützungsnotizen:

Zuverlässigste Methode, batteriebetriebene Geräte zu aktualisieren  
1. Platzieren Sie den Sensor in der Nähe von Homey (< 1 Meter)  
2. Ändern Sie die Einstellungen auf die gewünschten Werte  
3. Wecken Sie den Sensor auf (dreimal den Knopf drücken)  
4. Drücken Sie "Einstellungen speichern" während das Blinken der LED anzeigt, dass die Verbindung zu Homey hergestellt wird.

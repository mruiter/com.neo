Ondersteuningsnotities:

De meest betrouwbare manier om batterijgevoede apparaten bij te werken:  
1. Plaats de sensor dicht bij Homey (< 1 meter)  
2. Pas de instellingen aan naar de gewenste waarden  
3. Wek de sensor op (druk driemaal op de knop)  
4. Druk tijdens het knipperen van de LED (wat de connectie met Homey aangeeft) op "instellingen opslaan"

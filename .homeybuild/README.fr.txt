Notes de support :

Moyen le plus fiable pour mettre à jour les appareils à piles  
1. Placez le capteur près de Homey (< 1 mètre)  
2. Changez les paramètres selon vos souhaits  
3. Réveillez le capteur (cliquez trois fois sur le bouton)  
4. Pendant que la LED clignote (indiquant la connexion à Homey), appuyez sur "enregistrer les paramètres"
